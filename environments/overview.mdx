One of the primary benefits of using Architect is that **anyone** can create a new environment for
**any** application. Architect Components are fully portable and get deployed with everything they
need to run - even the APIs and databases they depend on.

There are many different ways to create environments with Architect. Click on one of the links below
to pick the strategy that works best for you.
