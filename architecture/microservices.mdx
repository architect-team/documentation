## Using dependencies

<CodeGroup>

```yaml Backend
name: backend

services:
  api:
    image: nginx:latest
    interfaces:
      main:
        port: 8080
```

```yaml Frontend
name: frontend

dependencies:
  backend: {}

services:
  app:
    image: nginx:latest
    interfaces:
      main:
        port: 8080
    environments:
      BACKEND_ADDR: ${{ dependencies.backend.services.api.interfaces.main.url }}
```
</CodeGroup>

## Debugging multiple components

```sh
$ architect link ./component-b
$ architect dev ./component-a
```

## Publishing to the registry

```sh
$ architect register ./component-a --tag v1 --account my-account
$ architect register ./component-b --tag v1 --account my-account
```
